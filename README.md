## Dissimilarity Adaptive CV in R

(Preliminary) R-Implementation of the dissimilarity-adaptive cross-validation (DA-CV) proposed by [Wang et al. (2024)](https://doi.org/10.1016/j.ecoinf.2025.103287).
The original Python-code has been made available by Wang et al. on [Zenodo](https://zenodo.org/records/13318611).
